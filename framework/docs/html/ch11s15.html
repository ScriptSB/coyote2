Metoda view()

Metoda tworzy instancję klasy View i zwraca obiekt tej klasy. Tak więc chcąc wyświetlić dany widok, możemy użyć następujących instrukcji: 

$view = new View('foo');
echo $view;
// albo:
$view = $this->load->view('foo');
echo $view;
