Wymagania sprzętowe

Jak każda aplikacja PHP, Coyote do działania potrzebuje interpretera PHP, w wersji 5.1 (zalecana wersja 5.3.3). Przydatny może okazać się również serwer ;) Polecamy popularny serwer Apache 2.0.X (aczkolwiek projekt testowany był również w wersji 2.2.X). Jeżeli Twoja aplikacja w domyśle ma korzystać z bazy danych, musisz zainstalować oraz skonfigurować odpowiedni system bazodanowy.

WAŻNE

Do wersji 1.1, Coyote Framework obsługuje jedynie bazę MySQL oraz PostgreSQL. Jeżeli chcesz obsługiwać w swojej aplikacji inną bazę, musisz napisać odpowiednią klasę obsługi danych (adapter). Więcej szczegołów znajdziesz w dalszej części podręcznika.
